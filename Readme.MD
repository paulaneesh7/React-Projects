## Installation

* Clone the repository
  
    ```
    git clone https://github.com/username/project.git
    ```
    
* Change into project directory
  
  ```
  cd project-number
  ```

* Install the dependencies
  
  ```
  npm i
  ```

* Run the App
  
  ```
  npm run dev
  ```
